Boli alergice

  • Alergia alimentara este un răspuns anormal la un aliment, mediat de sistemul imun.

    Simptome: vărsături, diaree, dureri abdominale, urticarie);

  • Dermatita atopica este o afecţiune cutanata inflamatorie  cu evolutie cronica.

    Simptome: eczema pe: fata - obraji, plica cotului, abdomen, membre inferioare;

  • Rinita alergica este o inflamație alergica a mucoasei nazale.

    Simptome: strănutul, secreții nazale - rinoree, nas înfundat - obstrucții nazale;

  • Conjunctivita alergica este manifestarea reacției alergice la nivelul mucoasei oculare.

    Simptome: lăcrimare, ochii înroșiți mâncărimi la nivelul ochilor - prurit;

  • Astmul este inflamația cronica a cailor respiratorii inferioare.

    Simptome: tuse seaca, greutate in respiraţie - dispnee, șuierat - wheezing;

  • Alergia Medicamentoasa este un răspuns anormal la unele medicamente.

    Simptome: urticație, edem facial, criza de astm, soc anafilactic;

  • Alergii la venin de insecte - albina, viespe

    Simptome: reacţie alergica locala, urticarie, edem facial, criza de astm, soc anafilactic;

  • Dermatita de contact este o inflamație a tegumentului.

    Simptome: eczema locala;

  • Urticaria este o afecţiune alergica caracterizata prin modificări la nivelul tegumentului.

    Simptome: papule (pete), prurit (mâncărimi) - elemente papulo-eritematoase pruriginoase;

  • Anafilaxia este o reacție alergica generalizata a corpului, una dintre complicațiile cele mai grave care pune viata in pericol - soc anafilactic.
